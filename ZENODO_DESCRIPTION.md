# DOME Copilot Experiment: Metadata, Full Text, and Evaluation Data

## Dataset Overview
This dataset supports the research on "DOME Copilot", an AI-assisted workflow for curating machine learning experiment metadata from scientific publications. It contains the source scientific articles (PDFs and Supplementary Material), the extracted metadata (JSON), and the evaluation results comparing Human vs. AI performance.

## File & Directory Structure

### 1. Registry & Reference Data
*   **`DOME_Registry_Human_Reviews_258_20260205.json`**: The core reference dataset containing human-reviewed metadata annotations.

### 2. Copilot Output (AI Generated)
*   **`Copilot_Processed_Datasets_JSON/`**: 
    *   Contains the structured JSON metadata generated by the DOME Copilot pipeline.
    *   Includes initial and updated processing versions for both the **222 Registry Subset** and the **1012 Positive Dataset**.

### 3. Source Data Cohorts (Full Text & Supplementary)
The dataset includes full-text PDFs and supplementary material archives for the three main study cohorts.

*   **`Download_222_DOME_Registry_PMC_Full_Text_and_Supplementary/`**:
    *   **Cohort**: 222 articles from the DOME Registry (Source Truth).
    *   **Contents**: `DOME_Registry_PMC_PDFs/` (Full Text), `DOME_Registry_PMC_Supplementary/` (Supp. Data), `DOME_Registry_TSV_Files/` (Metadata logs).

*   **`Download_1012_Positive_PMC_Full_Text_and_Supplementary/`**:
    *   **Cohort**: 1012 "Positive" articles (Machine Learning relevant).
    *   **Contents**: `Positive_PMC_PDFs/`, `Positive_PMC_Supplementary/`, `Positive_PMC_TSV_Files/`.

*   **`Download_1012_Negative_PMC_Full_Text_and_Supplementary/`**:
    *   **Cohort**: 1012 "Negative" articles (Control set / Non-ML).
    *   **Contents**: `Negative_PMC_PDFs/`, `Negative_PMC_Supplementary/` (Matched subset of 1012 entries), `Negative_PMC_TSV_Files/`.

### 4. Human vs. AI Evaluation
*   **`Human_30_Copilot_vs_Human_Evaluations_Interface/`**:
    *   Data and results for the detailed comparative study on a random subset of 30 papers.
    *   **Key Files**: 
        *   `evaluation_results.tsv`: Raw scores and evaluations.
        *   `statistical_analysis_report.txt`: Statistical significance tests (T-tests, Binomial tests).
        *   `30_Evaluation_Source_JSONs_Human_and_Copilot_Including_PDFs/`: Input data for the evaluation.

## Notes
*   User-specific data (profiles, emails) has been excluded for privacy.
*   System files and tool scripts (Python scripts) are generally excluded from this data archive, except where necessary for context (like evaluation interfaces).
